System prompts

Each role's standing system prompt, applied via --append-system-prompt. Pre-filled with the hydra's built-in default — edit to customize, or clear a box to restore the default. Applies to newly spawned agents (running ones are unaffected).

Default models

Passed as --model when spawning each role's agent. Blank uses the Claude CLI's own default. Aliases like opus, sonnet and haiku work, as do full model IDs.
