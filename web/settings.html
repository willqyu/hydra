Default system prompts

Appended to each agent's built-in brief via --append-system-prompt — the harness's own instructions stay intact. Leave blank for no extra prompt. Applies to newly spawned agents (running ones are unaffected).

Default models

Passed as --model when spawning each role's agent. Blank uses the Claude CLI's own default. Aliases like opus, sonnet and haiku work, as do full model IDs.
